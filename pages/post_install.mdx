# Post-Install Setup Guide

After installion, customize and configure osTicket in the admin panel to make the most of it. Just log in using the admin credentials you created during the installation process.

<Callout>
  Users need admin privileges to access the admin panel.
</Callout>

![osTicket admin panel](/post-install_1-admin-panel.png)


## Email Setup

Route incoming emails to your support workflow. Send tickets to specific departments and set rules to assign their priority.

For detailed instructions see [Email Settings](https://docs.osticket.com/en/latest/Getting%20Started/Email%20Settings.html).

## Help Topics

Provide ticket categories to make forms easier to fill and to assign priority. Users will be able to choose from a limited number of Help Topics when creating new tickets.

Check out how to [set up Help Topics](https://docs.osticket.com/en/latest/Admin/Manage/Help%20Topic.html).

![osTicket help topics](/post-install_2-help-topic.png)

## Departments

Build departments to manage key aspects of support like ticket allocation, team schedules and SLAs.

Explore all the possible options to [configure Departments](https://docs.osticket.com/en/v1.17.4/Admin/Agents/Departments.html)

![osTicket departments](/post-install_3-departments.png)

## Staff Members

Add unlimited Agents to the system. Each one with its proper department, access levels and permissions. Every staff account can be of type ‘Admin’ or ‘Staff’.
